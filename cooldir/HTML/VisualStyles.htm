VisualStyles Enumerator

Description

A list of the support visual styles that Fusion 360 uses when rendering the model.

Defined in namespace "adsk.core" and the header file is "Core/CoreTypeDefs.h".

Methods

Name Value Description
ShadedVisualStyle 1 Shaded display style.
ShadedWithHiddenEdgesVisualStyle 2 Shaded with hidden edges displayed.
ShadedWithVisibleEdgesOnlyVisualStyle 3 Shaded with only the visible edges displayed.
WireframeVisualStyle 4 Wireframe display style.
WireframeWithHiddenEdgesVisualStyle 5 Wireframe with hidden edges displayed.
WireframeWithVisibleEdgesOnlyVisualStyle 6 Wireframe with only the visible edges displayed.
