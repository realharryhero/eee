Camera.isFitView Property

Parent Object: Camera

Description

If this property is true, when this camera is applied to a viewport it will modify the camera such that the entire model is displayed in the viewport. When getting a camera from a viewport this property is always initialized to false.

Defined in namespace "adsk.core" and the header file is "adsk/core/Application/Camera.h".

Syntax

  • Python
  • C++
"camera_var" is a variable referencing a Camera object.

# Get the value of the property.
propertyValue = camera_var.isFitView


# Set the value of the property.
camera_var.isFitView = propertyValue

#include <Core/Application/Camera.h>

// Get the value of the property.
boolean propertyValue = camera_var->isFitView();


// Set the value of the property, where value_var is a boolean.
bool returnValue = camera_var->isFitView(value_var);

Property Value

This is a read/write property whose value is a boolean.
