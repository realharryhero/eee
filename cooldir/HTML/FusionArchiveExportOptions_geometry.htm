FusionArchiveExportOptions.geometry Property

Parent Object: FusionArchiveExportOptions

Description

Specifies the geometry to export. This can be Occurrence, or the root Component. For STLExport, we also support BRepBody.

Defined in namespace "adsk.fusion" and the header file is "adsk/fusion/Fusion/FusionArchiveExportOptions.h".
