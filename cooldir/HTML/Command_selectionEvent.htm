Command.selectionEvent Event

Parent Object: Command

Description

Gets an event that is fired during the selection process. The event is fired as the user moves the mouse over the model but before anything is made selectable. The entity currently under the mouse is provided through the SelectionEventArgs and you can specify whether it is selectable or not. This provides a dynamic way of defining the selection filter.

Defined in namespace "adsk.core" and the header file is "adsk/core/UserInterface/Command.h".
