ExportManager.execute Method

Parent Object: ExportManager

Description

Executes the export operation to create the file in the format specified by the input ExportOptions object.

Defined in namespace "adsk.fusion" and the header file is "adsk/fusion/Fusion/ExportManager.h".

Syntax

  • Python
  • C++
"exportManager_var" is a variable referencing an ExportManager object.
returnValue = exportManager_var.execute(exportOptions)
"exportManager_var" is a variable referencing an ExportManager object.

#include <Fusion/Fusion/ExportManager.h>

returnValue = exportManager_var->execute(exportOptions);

Return Value

Type Description
boolean Returns true if the export was successful.

Parameters

Name Type Description
exportOptions ExportOptions An ExportOptions object that is created using one of the create methods on the ExportManager object. This defines the type of file and any available options supported for that file type.
