VariableRadiusFilletEdgeSet.edges Property

Parent Object: VariableRadiusFilletEdgeSet

Description

Gets and sets the edges that will be filleted. In order to access (get) the input edges of a fillet, you must roll the timeline back to just before the fillet feature whose edges you want to access.

Defined in namespace "adsk.fusion" and the header file is "adsk/fusion/Features/VariableRadiusFilletEdgeSet.h".
