FloatSliderCommandInput.setText Method

Parent Object: FloatSliderCommandInput

Description

Sets the text of the slider. Both the left and the right text should be set.

Defined in namespace "adsk.core" and the header file is "adsk/core/UserInterface/FloatSliderCommandInput.h".

Syntax

  • C++

returnValue = floatSliderCommandInput_var.setText(left, right)

#include <Core/UserInterface/FloatSliderCommandInput.h>

Return Value

Type Description
boolean Returns true if successful.

Parameters

Name Type Description
