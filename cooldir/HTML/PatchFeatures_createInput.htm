PatchFeatures.createInput Method

Parent Object: PatchFeatures

Description

Creates a PatchFeatureInput object. Use properties and methods on the returned PatchFeatureInput object to set other settings. The PatchFeatureInput object can then be used as input to the add method to create the patch feature.

Defined in namespace "adsk.fusion" and the header file is "adsk/fusion/Features/PatchFeatures.h".

Syntax

  • Python
  • C++
"patchFeatures_var" is a variable referencing a PatchFeatures object.
returnValue = patchFeatures_var.createInput(boundaryCurve, operation)
"patchFeatures_var" is a variable referencing a PatchFeatures object.

#include <Fusion/Features/PatchFeatures.h>

returnValue = patchFeatures_var->createInput(boundaryCurve, operation);

Return Value

Type Description
PatchFeatureInput Returns the newly created PatchFeatureInput object or null if the creation failed.

Parameters

Name Type Description
boundaryCurve Base Defines the input geometry that will be used to define the boundary. This can be a sketch profile, a single sketch curve, a single B-Rep edge, or a Path object.

If a single sketch curve or B-Rep edge is input, that is not closed, Fusion 360 will automatically find connected sketch curves or B-Rep edges in order to define a closed loop. All sketch curves are valid as input. BRepEdges are valid if they are an "open" edge, which means they are only used by one face.

If a Path is input it must define a closed shape and the B-Rep edges must be valid "open" edges.
operation FeatureOperations The feature operation to perform. Only 'NewBodyFeatureOperation' and 'NewComponentFeatureOperation' are valid operations for patch features.
