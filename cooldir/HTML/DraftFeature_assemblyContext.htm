DraftFeature.assemblyContext Property

Parent Object: DraftFeature

Description

Returns the assembly occurrence (i.e. the occurrence) of this object in an assembly. This is only valid in the case where this is acting as a proxy in an assembly. Returns null in the case where the object is not in the context of an assembly but is already the native object.

Defined in namespace "adsk.fusion" and the header file is "adsk/fusion/Features/DraftFeature.h".

Syntax

  • Python
  • C++
"draftFeature_var" is a variable referencing a DraftFeature object.

# Get the value of the property.
propertyValue = draftFeature_var.assemblyContext
"draftFeature_var" is a variable referencing a DraftFeature object.
#include <Fusion/Features/DraftFeature.h>

// Get the value of the property.
Ptr<Occurrence> propertyValue = draftFeature_var->assemblyContext();

Property Value

This is a read only property whose value is an Occurrence.
