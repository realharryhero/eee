DraftFeature.setSingleAngle Method

Parent Object: DraftFeature

Description

Changes the definition of the feature so that a single angle is used for all drafts. If the isSymmetric is true then the faces are split along the parting plane and drafted independently using the same angle.

Defined in namespace "adsk.fusion" and the header file is "adsk/fusion/Features/DraftFeature.h".

Syntax

  • Python
  • C++
"draftFeature_var" is a variable referencing a DraftFeature object.
returnValue = draftFeature_var.setSingleAngle(isSymmetric, angle)
"draftFeature_var" is a variable referencing a DraftFeature object.

#include <Fusion/Features/DraftFeature.h>

returnValue = draftFeature_var->setSingleAngle(isSymmetric, angle);

Return Value

Type Description
boolean Returns true if successful

Parameters

Name Type Description
isSymmetric boolean Set to 'true' if the faces are to be split along the plane and drafted symmetrically. This will have the side effect of setting the isSymmetric property to the same value.
angle ValueInput The ValueInput object that defines the angle of the draft. This can be a positive or negative value which will affect the direction of the draft along with the isDirectionFlipped property.
