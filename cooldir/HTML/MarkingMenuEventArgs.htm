MarkingMenuEventArgs Object

Derived from: EventArgs Object

Description

The MarkingMenuEventArgs provides information associated with the marking and context menu being displayed.

Defined in namespace "adsk.core" and the header file is "Core/UserInterface/MarkingMenuEventArgs.h".

Methods

Name Description
classType Static function that all classes support that returns the type of the class as a string. The returned string matches the string returned by the objectType property. For example if you have a reference to an object and you want to check if it's a SketchLine you can use myObject.objectType == fusion.SketchLine.classType().

Properties

Name Description
firingEvent The event that the firing is in response to.
isValid Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.
linearMarkingMenu Provides access to the linear marking menu.
objectType Returns a string indicating the type of the object.
radialMarkingMenu Provides access to the radial marking menu.
selectedEntities Returns the currently selected entities that the user left-clicked over. These provide the "context" of what should be displayed in the menu. This can be an empty array in the case where they clicked in a open area within the graphics window.
