CustomGraphicsSolidColorEffect.color Property

Parent Object: CustomGraphicsSolidColorEffect

Description

The color to use for the solid color display. The opacity component of the color is ignored because the opacity of custom graphics is controlled seperately using an opacity attribute.

Defined in namespace "adsk.fusion" and the header file is "adsk/fusion/Graphics/CustomGraphicsSolidColorEffect.h".
