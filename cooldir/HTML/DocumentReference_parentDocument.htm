DocumentReference.parentDocument Property

Parent Object: DocumentReference

Description

The document that is doing the referencing and owns this reference.

Defined in namespace "adsk.core" and the header file is "adsk/core/Application/DocumentReference.h".

Syntax

  • Python
  • C++
"documentReference_var" is a variable referencing a DocumentReference object.

# Get the value of the property.
propertyValue = documentReference_var.parentDocument
"documentReference_var" is a variable referencing a DocumentReference object.
#include <Core/Application/DocumentReference.h>

// Get the value of the property.
Ptr<Document> propertyValue = documentReference_var->parentDocument();

Property Value

This is a read only property whose value is a Document.
