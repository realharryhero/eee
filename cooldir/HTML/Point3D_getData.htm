Point3D.getData Method

Parent Object: Point3D

Description

Gets the data defining the point.

Defined in namespace "adsk.core" and the header file is "adsk/core/Geometry/Point3D.h".

Syntax

  • Python
  • C++
"point3D_var" is a variable referencing a Point3D object.

(returnValue, x, y, z) = point3D_var.getData()
 

#include <Core/Geometry/Point3D.h

// Declare the output arguments.
double x;
double y;
double z;
boolean returnValue = point3D_var->getData(x, y, z);

Return Value

Type Description
boolean Returns true if successful.

Parameters

Name Type Description
x double The output x coordinate of the point.
y double The output y coordinate of the point.
z double The output z coordinate of the point.
