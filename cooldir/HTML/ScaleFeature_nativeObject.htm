ScaleFeature.nativeObject Property

Parent Object: ScaleFeature

Description

The NativeObject is the object outside the context of an assembly and in the context of it's parent component. Returns null in the case where this object is not in the context of an assembly but is already the native object.

Defined in namespace "adsk.fusion" and the header file is "adsk/fusion/Features/ScaleFeature.h".

Syntax

  • Python
  • C++
"scaleFeature_var" is a variable referencing a ScaleFeature object.

# Get the value of the property.
propertyValue = scaleFeature_var.nativeObject
"scaleFeature_var" is a variable referencing a ScaleFeature object.
#include <Fusion/Features/ScaleFeature.h>

// Get the value of the property.
Ptr<ScaleFeature> propertyValue = scaleFeature_var->nativeObject();

Property Value

This is a read only property whose value is a ScaleFeature.
