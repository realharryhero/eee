IntegerProperty Object

Derived from: Property Object

Description

An integer value property associated with a material or appearance.

Defined in namespace "adsk.core" and the header file is "Core/Materials/IntegerProperty.h".

Methods

Name Description
classType Static function that all classes support that returns the type of the class as a string. The returned string matches the string returned by the objectType property. For example if you have a reference to an object and you want to check if it's a SketchLine you can use myObject.objectType == fusion.SketchLine.classType().
getLimits Method that returns any limits for the value of this property. The HasLimits property can be used to see if there are any limits or not.

Properties

Name Description
hasLimits Gets the boolean flag that indicates if the value of this property has any limits it must be within to be valid. If True, use the GetLimits method to get the limit values.
hasMultipleValues Gets the boolean flag that indicates if this property has multiple values or not.
id Returns the unique ID of this property.
isReadOnly Indicates if this property is read-only. If True any attempted edits will fail.
isValid Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.
name Returns the name of this property as seen in the user interface. This name is localized and can change based on the current language
objectType Returns a string indicating the type of the object.
parent Returns the parent of this property.
value Gets and sets this property value. The value of this property should be ignored if the HasConnectedTexture property is true. Setting this will remove any associated texture, if there is one.
values Gets and sets the values associated with this property. HasMultipleValues property indicates if this property will be returning more than one value.
