UserInterface.allToolbarPanels Property

Parent Object: UserInterface

Description

Gets all of the toolbar panels. This returns all of the panels available, regardless of which workspace or product they're associated with.

Defined in namespace "adsk.core" and the header file is "adsk/core/UserInterface/UserInterface.h".

Syntax

  • Python
  • C++
"userInterface_var" is a variable referencing a UserInterface object.

# Get the value of the property.
propertyValue = userInterface_var.allToolbarPanels
"userInterface_var" is a variable referencing a UserInterface object.
#include <Core/UserInterface/UserInterface.h>

// Get the value of the property.
Ptr<ToolbarPanelList> propertyValue = userInterface_var->allToolbarPanels();

Property Value

This is a read only property whose value is a ToolbarPanelList.
