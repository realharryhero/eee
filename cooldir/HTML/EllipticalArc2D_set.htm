EllipticalArc2D.set Method

Parent Object: EllipticalArc2D

Description

Sets all of the data defining the elliptical arc. A Point2D object that defines the center of the elliptical arc. The major axis of the elliptical arc. The major radius of the of the elliptical arc. The minor radius of the of the elliptical arc. The start angle of the elliptical arc in radians, where 0 is along the major axis. The end angle of the elliptical arc in radians, where 0 is along the major axis. Returns true if redefining the elliptical arc is successful
