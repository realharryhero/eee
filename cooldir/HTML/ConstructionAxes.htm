ConstructionAxes Object

Derived from: Base Object

Description

Provides access to the construction axes within a component and provides methods to create new construction axes.

Defined in namespace "adsk.fusion" and the header file is "Fusion/Construction/ConstructionAxes.h".

Methods

Name Description
add Creates and adds a new ConstructionAxis using the creation parameters in the ConstructionAxisInput. If the ConstructionAxisInput was defined using the setByLine method then the add will only work in a direct edit model (do not capture design history) and will fail in a parametric model.
classType Static function that all classes support that returns the type of the class as a string. The returned string matches the string returned by the objectType property. For example if you have a reference to an object and you want to check if it's a SketchLine you can use myObject.objectType == fusion.SketchLine.classType().
createInput Create a ConstructionAxisInput object that is in turn used to create a ConstructionAxis.
item Function that returns the specified construction axis using an index into the collection.
itemByName Returns the specified construction axis using the name of the construction axis as it is displayed in the browser.

Properties

Name Description
component The component that owns this collection.
count The number of construction axes in the collection.
isValid Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.
objectType Returns a string indicating the type of the object.
