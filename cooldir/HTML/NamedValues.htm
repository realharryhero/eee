NamedValues Object

Derived from: Base Object

Description

Wraps a list of named values.

Defined in namespace "adsk.core" and the header file is "Core/Application/NamedValues.h".

Methods

Name Description
add Adds a name value pair to the NamedValues object
classType Static function that all classes support that returns the type of the class as a string. The returned string matches the string returned by the objectType property. For example if you have a reference to an object and you want to check if it's a SketchLine you can use myObject.objectType == fusion.SketchLine.classType().
create Creates a transient NamedValues object.
getByIndex Function that returns the name and ValueInput object of a name value pair by specifying an index number
getValueByName Function that returns the ValueInput object of a name value pair by specifying its name

Properties

Name Description
count Returns the number of name value pairs in this object.
isValid Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.
objectType Returns a string indicating the type of the object.

Accessed From
