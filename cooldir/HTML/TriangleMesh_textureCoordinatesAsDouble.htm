TriangleMesh.textureCoordinatesAsDouble Property

Parent Object: TriangleMesh

Description

Returns the texture coordinates used when mapping a texture to this face. The coordinates are returned as an array of doubles where they are the u and v components of each coordinate as defined in parametric space. There is a texture coordinate for each vertex in the face mesh.

Defined in namespace "adsk.fusion" and the header file is "adsk/fusion/MeshData/TriangleMesh.h".

Syntax

# Get the value of the property.
propertyValue = triangleMesh_var.textureCoordinatesAsDouble

Property Value

This is a read only property whose value is an array of type Double.
