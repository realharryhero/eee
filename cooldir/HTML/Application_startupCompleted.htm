Application.startupCompleted Event

Parent Object: Application

Description

The startupCompleted event fires after Fusion 360 has completed its initialization. This includes initialization of all the Add-ins loaded at startup. The client can add or remove ApplicationEventHandlers from the ApplicationEvent.

Defined in namespace "adsk.core" and the header file is "adsk/core/Application/Application.h".

Syntax

  • Python
  • C++
-------- Global variables ---------
# Global variable used to maintain a reference to all event handlers.
handlers = []


-------- Event handler class definition ---------
# Event handler for the startupCompleted event.
class MyStartupCompletedHandler(adsk.core.ApplicationEventHandler):
    def __init__(self):
        super().__init__()
    def notify(self, args):
        eventArgs = adsk.core.ApplicationEventArgs.cast(args)

        # Code to react to the event.
        ui.messageBox('In MyStartupCompletedHandler event handler.')


-------- Connect the handler to the event. ---------
# "application_var" is a variable referencing an Application object.
onStartupCompleted = MyStartupCompletedHandler()
application_var.startupCompleted.add(onStartupCompleted)
handlers.append(onStartupCompleted)
--------- Required include files. ---------
#include <Core/Application/Application.h>

#include <Core/Application/ApplicationEventHandler.h>

--------- Event handler class definition and global declaration. ---------
// Event handler for the startupCompleted event.
class MyStartupCompletedEventHandler : public adsk::core::ApplicationEventHandler
{
public:
    void notify(const Ptr<ApplicationEventArgs>& eventArgs) override
    {
        // Code to react to the event.
        ui.messageBox("In MyStartupCompletedEventHandler event handler.");
    }
} _startupCompleted;


--------- Connect the handler to the event. ---------
// "application_var" is a variable referencing an Application object.
// Connect the handler function to the event.
Ptr<ApplicationEvent> startupCompletedEvent = application_var->startupCompleted();
if (!startupCompletedEvent)
    return;

bool isOk = startupCompletedEvent->add(&_startupCompleted);
if (!isOk)
    return;

Property Value

This is an event property that returns an ApplicationEvent.
