Ellipse2D.getData Method

Parent Object: Ellipse2D

Description

Gets all of the data defining the ellipse.

Defined in namespace "adsk.core" and the header file is "adsk/core/Geometry/Ellipse2D.h".

Syntax

  • Python
  • C++
"ellipse2D_var" is a variable referencing an Ellipse2D object.

(returnValue, center, majorAxis, majorRadius, minorRadius) = ellipse2D_var.getData()
 

#include <Core/Geometry/Ellipse2D.h

// Declare the output arguments.
Ptr<Point2D> center;
Ptr<Vector2D> majorAxis;
double majorRadius;
double minorRadius;
boolean returnValue = ellipse2D_var->getData(center, majorAxis, majorRadius, minorRadius);

Return Value

Type Description
boolean Returns true if successful.

Parameters

Name Type Description
center Point2D The output center point of the ellipse.
majorAxis Vector2D The output major axis of the ellipse.
majorRadius double The output major radius of the of the ellipse.
minorRadius double The output minor radius of the of the ellipse.
