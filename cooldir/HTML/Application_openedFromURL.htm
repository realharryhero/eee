Application.openedFromURL Event

Parent Object: Application

Description

The openedFromURL event fires after the user has clicked a link in a web page that uses the Fusion 360 protocol handler to create a new using an existing file as the initial contents and that operation has completed.

Defined in namespace "adsk.core" and the header file is "adsk/core/Application/Application.h".
