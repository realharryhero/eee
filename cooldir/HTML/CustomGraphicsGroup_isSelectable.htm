CustomGraphicsGroup.isSelectable Property

Parent Object: CustomGraphicsGroup

Description

Gets and sets if the graphics entity is selectable within the graphics window. By default, when a new entity is created it is selectable.

Defined in namespace "adsk.fusion" and the header file is "adsk/fusion/Graphics/CustomGraphicsGroup.h".

Syntax

# Get the value of the property.
propertyValue = customGraphicsGroup_var.isSelectable


# Set the value of the property.
customGraphicsGroup_var.isSelectable = propertyValue
