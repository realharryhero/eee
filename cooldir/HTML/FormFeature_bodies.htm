FormFeature.bodies Property

Parent Object: FormFeature

Description

Returns the bodies that were modified or created by this feature. This works for both parametric and non-parametric features. For a BaseFeature this returns the bodies that are owned by the base feature.

Defined in namespace "adsk.fusion" and the header file is "adsk/fusion/Features/FormFeature.h".

Syntax

  • Python
  • C++
"formFeature_var" is a variable referencing a FormFeature object.

# Get the value of the property.
propertyValue = formFeature_var.bodies
"formFeature_var" is a variable referencing a FormFeature object.
#include <Fusion/Features/FormFeature.h>

// Get the value of the property.
Ptr<BRepBodies> propertyValue = formFeature_var->bodies();

Property Value

This is a read only property whose value is a BRepBodies.
