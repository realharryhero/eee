DocumentEventArgs Object

Derived from: EventArgs Object

Description

The DocumentEventArgs provides information associated with a document event. Note that some properties are not available on every event - for example, the Document is not available on the DocumentOpening event because the Document is not yet available.

Defined in namespace "adsk.core" and the header file is "Core/Application/DocumentEventArgs.h".

Methods

Name Description
classType Static function that all classes support that returns the type of the class as a string. The returned string matches the string returned by the objectType property. For example if you have a reference to an object and you want to check if it's a SketchLine you can use myObject.objectType == fusion.SketchLine.classType().

Properties

Name Description
document Provides access to the document that is open. Can be null in the case where the event is fired before the document has been opened or after it has been closed.
firingEvent The event that the firing is in response to.
fullPath The full path to the file.
isValid Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.
objectType Returns a string indicating the type of the object.
