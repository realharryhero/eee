ImportOptions Object

Derived from: Base Object

Description

The base class for the different import types. This class is never directly used in an import because you need the specific import type to specify the type of import to be performed.

Defined in namespace "adsk.core" and the header file is "Core/Application/ImportOptions.h".

Methods

Name Description
classType Static function that all classes support that returns the type of the class as a string. The returned string matches the string returned by the objectType property. For example if you have a reference to an object and you want to check if it's a SketchLine you can use myObject.objectType == fusion.SketchLine.classType().

Properties

Name Description
filename Gets and sets the filename or url of the file to be imported.
isValid Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.
isViewFit Specifies if the camera should be adjusted to fit the geometry of the import. This defaults to true, which will cause a change in the current view. Setting this to false will leave the view as-is and just import the geometry.
objectType Returns a string indicating the type of the object.
