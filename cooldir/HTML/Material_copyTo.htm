Material.copyTo Method

Parent Object: Material

Description

Copies this material to the specified target. ***Depricated Method: Use the addByCopyMethod on the Materials object instead, where you can rename the copied Material***

Defined in namespace "adsk.core" and the header file is "adsk/core/Materials/Material.h".

Syntax

  • Python
  • C++
"material_var" is a variable referencing a Material object.
returnValue = material_var.copyTo(target)
"material_var" is a variable referencing a Material object.

#include <Core/Materials/Material.h>

returnValue = material_var->copyTo(target);

Return Value

Type Description
Material Returns the new copy of the material or null if the copy failed.

Parameters

Name Type Description
target Base The target can be a Design or MaterialFavorites object.
