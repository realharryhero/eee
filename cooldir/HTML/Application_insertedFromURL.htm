Application.insertedFromURL Event

Parent Object: Application

Description

The insertedFromURL event fires after the user has clicked a link in a web page that uses the Fusion 360 protocol handler to insert a file as new component and that operation has completed.

Defined in namespace "adsk.core" and the header file is "adsk/core/Application/Application.h".
