MoveFeatureInput.transform Property

Parent Object: MoveFeatureInput

Description

Gets and sets the tranform to apply to the input entities. This can describe a move (translation) or a rotation. The matrix must define an orthogonal transform. That is the axes remain perpendicular to each other and there isn't any scale or mirror defined.

Defined in namespace "adsk.fusion" and the header file is "adsk/fusion/Features/MoveFeatureInput.h".
