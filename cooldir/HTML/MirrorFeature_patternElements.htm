MirrorFeature.patternElements Property

Parent Object: MirrorFeature

Description

Gets the PatternElements collection that contains the elements created by this pattern.

Defined in namespace "adsk.fusion" and the header file is "adsk/fusion/Features/MirrorFeature.h".
