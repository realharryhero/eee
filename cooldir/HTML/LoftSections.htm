LoftSections Object

Derived from: Base Object

Description

The set of two or more sections used to define the shape of the loft.

Defined in namespace "adsk.fusion" and the header file is "Fusion/Features/LoftSections.h".

Methods

Name Description
add Adds a new section to the loft. The initial end condition is "Free". Additional methods on the returned LoftSection can be used to further define the section.
classType Static function that all classes support that returns the type of the class as a string. The returned string matches the string returned by the objectType property. For example if you have a reference to an object and you want to check if it's a SketchLine you can use myObject.objectType == fusion.SketchLine.classType().
item Function that returns the specified LoftSection using an index into the collection. They are returned in the same order that they are used in the loft. Their order can be modified using the reorder method of the LoftSection object.

Properties

Name Description
count The number of LoftSections in the collection.
isValid Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.
objectType Returns a string indicating the type of the object.
