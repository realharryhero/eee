Design.userParameters Property

Parent Object: Design

Description

Returns the collection of User Parameters in a design

Defined in namespace "adsk.fusion" and the header file is "adsk/fusion/Fusion/Design.h".

Syntax

  • Python
  • C++
"design_var" is a variable referencing a Design object.

# Get the value of the property.
propertyValue = design_var.userParameters
"design_var" is a variable referencing a Design object.
#include <Fusion/Fusion/Design.h>

// Get the value of the property.
Ptr<UserParameters> propertyValue = design_var->userParameters();

Property Value

This is a read only property whose value is a UserParameters.

Samples

Name Description
Set parameters from a csv file and export to STEP Reads data from a .csv file and sets user parameters in the model and then exports the model to STEP. When setting parameters be aware that this sample is setting user parameters. It's also possible to set model parameters but that's not demonstrated here. Also when accessing parameters, it is case sensitive so the names you use in your program much exactly match the names in the model.
