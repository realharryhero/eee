ThreadFeature.isSuppressed Property

Parent Object: ThreadFeature

Description

Gets and sets if this feature is suppressed. This is only valid for parametric features.

Defined in namespace "adsk.fusion" and the header file is "adsk/fusion/Features/ThreadFeature.h".

Syntax

  • Python
  • C++
"threadFeature_var" is a variable referencing a ThreadFeature object.

# Get the value of the property.
propertyValue = threadFeature_var.isSuppressed


# Set the value of the property.
threadFeature_var.isSuppressed = propertyValue
