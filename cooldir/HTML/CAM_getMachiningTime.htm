CAM.getMachiningTime Method

Parent Object: CAM

Description

Get the machining time for the specified objects.

Defined in namespace "adsk.cam" and the header file is "adsk/cam/CAM/CAM.h".

Syntax

  • Python
  • C++
"cAM_var" is a variable referencing a CAM object.
returnValue = cAM_var.getMachiningTime(operations, feedScale, rapidFeed, toolChangeTime)
"cAM_var" is a variable referencing a CAM object.

#include <Cam/CAM/CAM.h>

returnValue = cAM_var->getMachiningTime(operations, feedScale, rapidFeed, toolChangeTime);

Return Value

Type Description
MachiningTime Returns a MachiningTime object that has properties holding the calculation results.

Parameters

Name Type Description
operations Base An Operation, Setup, Folder or Pattern object for which to get the machining time for. You can also specify a collection of any combination of these object types.
feedScale double The feed scale value (%) to use.
rapidFeed double The rapid feed rate in centimeters per second.
toolChangeTime double The tool change time in seconds.

Introduced in version September 2017
