Color Object

Derived from: Base Object

Description

The Color class wraps all of the information that defines a simple color.

Defined in namespace "adsk.core" and the header file is "Core/Application/Color.h".

Methods

Name Description
classType Static function that all classes support that returns the type of the class as a string. The returned string matches the string returned by the objectType property. For example if you have a reference to an object and you want to check if it's a SketchLine you can use myObject.objectType == fusion.SketchLine.classType().
create Creates a new color.
getColor Gets all of the information defining this color.
setColor Sets all of the color information.

Properties

Name Description
blue Gets and sets the blue component of the color. The value can be 0 to 255.
green Gets and sets the green component of the color. The value can be 0 to 255.
isValid Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.
objectType Returns a string indicating the type of the object.
opacity Gets and sets the opacity of the color. The value can be 0 to 255.
red Gets and sets the red component of the color. The value can be 0 to 255.
