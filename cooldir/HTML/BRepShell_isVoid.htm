BRepShell.isVoid Property

Parent Object: BRepShell

Description

Returns true if the faces of this shell bound a void or an empty space within an outer shell.

Defined in namespace "adsk.fusion" and the header file is "adsk/fusion/BRep/BRepShell.h".
