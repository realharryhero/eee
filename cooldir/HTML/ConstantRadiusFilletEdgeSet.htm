ConstantRadiusFilletEdgeSet Object

Derived from: FilletEdgeSet Object

Description

Provides access to the edges and the parameter associated with a constant radius fillet.

Defined in namespace "adsk.fusion" and the header file is "Fusion/Features/ConstantRadiusFilletEdgeSet.h".

Methods

Name Description
classType Static function that all classes support that returns the type of the class as a string. The returned string matches the string returned by the objectType property. For example if you have a reference to an object and you want to check if it's a SketchLine you can use myObject.objectType == fusion.SketchLine.classType().

Properties

Name Description
edges Gets and sets the edges that will be filleted. In order to access (get) the input edges of a fillet, you must roll the timeline back to just before the fillet feature whose edges you want to access.
isValid Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.
objectType Returns a string indicating the type of the object.
radius Returns the model parameter that controls the radius of the fillet. You can edit the radius by using the properties on the returned ModelParameter object.
