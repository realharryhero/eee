SketchEllipse.split Method

Parent Object: SketchEllipse

Description

Split a curve at a position specified along the curve

Defined in namespace "adsk.fusion" and the header file is "adsk/fusion/Sketch/SketchEllipse.h".

Syntax

  • Python
  • C++
"sketchEllipse_var" is a variable referencing a SketchEllipse object.
# Uses no optional arguments.
returnValue = sketchEllipse_var.split(splitPoint)

# Uses optional arguments.
returnValue = sketchEllipse_var.split(splitPoint, createConstraints)
"sketchEllipse_var" is a variable referencing a SketchEllipse object.

#include <Fusion/Sketch/SketchEllipse.h>

// Uses no optional arguments.
returnValue = sketchEllipse_var->split(splitPoint);

// Uses optional arguments.
returnValue = sketchEllipse_var->split(splitPoint, createConstraints);

Return Value

Type Description
ObjectCollection Returns the resulting 2 curves; the orginal curve + the newly created curve When split spline the original is deleted and two new curves returned. Empty collection returned if curve is closed.

Parameters

Name Type Description
splitPoint Point3D A position (transient Point3D) on the curve that defines the point at which to split the curve
createConstraints boolean Constraints are created by default. Specify false to create no constraints.

This is an optional argument whose default value is True.
