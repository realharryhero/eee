Command.mouseDrag Event

Parent Object: Command

Description

Gets an event that is fired when the mouse is in drag mode, (being moved while a button is pressed).

Defined in namespace "adsk.core" and the header file is "adsk/core/UserInterface/Command.h".

Syntax

  • C++
-------- Global variables ---------
# Global variable used to maintain a reference to all event handlers.
handlers = []


-------- Event handler class definition ---------
# Event handler for the mouseDrag event.
class MyMouseDragHandler(adsk.core.MouseEventHandler):
    def __init__(self):
        super().__init__()
    def notify(self, args):
        eventArgs = adsk.core.MouseEventArgs.cast(args)

        # Code to react to the event.
        ui.messageBox('In MyMouseDragHandler event handler.')


-------- Connect the handler to the event. ---------
# "command_var" is a variable referencing a Command object.
onMouseDrag = MyMouseDragHandler()

--------- Event handler class definition and global declaration. ---------
// Event handler for the mouseDrag event.
class MyMouseDragEventHandler : public adsk::core::MouseEventHandler
{
public:
    void notify(const Ptr<MouseEventArgs>& eventArgs) override
    {
        // Code to react to the event.
        ui.messageBox("In MyMouseDragEventHandler event handler.");
    }
} _mouseDrag;


--------- Connect the handler to the event. ---------
// "command_var" is a variable referencing a Command object.
// Connect the handler function to the event.
Ptr<MouseEvent> mouseDragEvent = command_var->mouseDrag();
if (!mouseDragEvent)
    return;

bool isOk = mouseDragEvent->add(&_mouseDrag);
if (!isOk)
    return;

Property Value

This is an event property that returns a MouseEvent.
