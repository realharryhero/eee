SplitButtonControl.defaultCommandDefinition Property

Parent Object: SplitButtonControl

Description

Gets the command definition that is used as the default command on the main portion of the split button.

Defined in namespace "adsk.core" and the header file is "adsk/core/UserInterface/SplitButtonControl.h".

Syntax

  • Python
  • C++
"splitButtonControl_var" is a variable referencing a SplitButtonControl object.

# Get the value of the property.
propertyValue = splitButtonControl_var.defaultCommandDefinition
"splitButtonControl_var" is a variable referencing a SplitButtonControl object.
#include <Core/UserInterface/SplitButtonControl.h>

// Get the value of the property.
Ptr<CommandDefinition> propertyValue = splitButtonControl_var->defaultCommandDefinition();

Property Value

This is a read only property whose value is a CommandDefinition.
