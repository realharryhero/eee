CutPasteBodies.item Method

Parent Object: CutPasteBodies

Description

Function that returns the specified Cut/Paste Body feature using an index into the collection.

Defined in namespace "adsk.fusion" and the header file is "adsk/fusion/Features/CutPasteBodies.h".

Type Description
