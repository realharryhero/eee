ConstructionPlaneThreePointsDefinition Object

Derived from: ConstructionPlaneDefinition Object

Description

ConstructionPlaneThreePointDefinition defines a ConstructionPlane by 3 point entities (e.g. (sketch points, vectrices or construction points) that form a triangle (i.e. no two points the same and they aren't collinear).

Defined in namespace "adsk.fusion" and the header file is "Fusion/Construction/ConstructionPlaneThreePointsDefinition.h".

Methods

Name Description
classType Static function that all classes support that returns the type of the class as a string. The returned string matches the string returned by the objectType property. For example if you have a reference to an object and you want to check if it's a SketchLine you can use myObject.objectType == fusion.SketchLine.classType().
redefine Redefines the input geometry of the construction plane.
