TorusFeature.baseFeature Property

Parent Object: TorusFeature

Description

If this feature is associated with a base feature, this property will return that base feature. If it's not associated with a base feature, this property will return null.

Defined in namespace "adsk.fusion" and the header file is "adsk/fusion/Features/TorusFeature.h".

Syntax

  • Python
  • C++
"torusFeature_var" is a variable referencing a TorusFeature object.

# Get the value of the property.
propertyValue = torusFeature_var.baseFeature
"torusFeature_var" is a variable referencing a TorusFeature object.
#include <Fusion/Features/TorusFeature.h>

// Get the value of the property.
Ptr<BaseFeature> propertyValue = torusFeature_var->baseFeature();

Property Value

This is a read only property whose value is a BaseFeature.
