SketchTexts Object

Derived from: Base Object

Description

The collection of text blocks in a sketch. This provides access to the existing text blocks and supports creating new text blocks.

Defined in namespace "adsk.fusion" and the header file is "Fusion/Sketch/SketchTexts.h".

Methods

Name Description
add Creates a sketch text.
classType Static function that all classes support that returns the type of the class as a string. The returned string matches the string returned by the objectType property. For example if you have a reference to an object and you want to check if it's a SketchLine you can use myObject.objectType == fusion.SketchLine.classType().
createInput Creates a SketchTextInput object that can be used to define additional settings when creating sketch text. The SketchTextInput object is equivalent to the Sketch Text dialog in that it collects all of the input required to create sketch text. Once the properties of the SketchTextInput object have been defined, use the add method to create the sketch text.
item Function that returns the specified sketch text using an index into the collection.

Properties

Name Description
