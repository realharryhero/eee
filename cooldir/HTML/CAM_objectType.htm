CAM.objectType Property

Parent Object: CAM

Description

Returns a string indicating the type of the object.

Defined in namespace "adsk.cam" and the header file is "adsk/cam/CAM/CAM.h".

Syntax

  • Python
  • C++
"cAM_var" is a variable referencing a CAM object.

# Get the value of the property.
propertyValue = cAM_var.objectType
"cAM_var" is a variable referencing a CAM object.
#include <Cam/CAM/CAM.h>

// Get the value of the property.
string propertyValue = cAM_var->objectType();

Property Value

This is a read only property whose value is a string.
