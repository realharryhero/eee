MeshBodies Object

Derived from: Base Object

Description

Provides access to the MeshBodies in the parent Component and supports the creation of new mesh bodies.

Defined in namespace "adsk.fusion" and the header file is "Fusion/MeshBody/MeshBodies.h".

Methods

Name Description
add Creates a new mesh body by importing an .stl or .obj file. Because of a current limitation, if you want to create a mesh body in a parametric model, you must first call the edit method of the base or form feature, use this method to create the mesh body, and then call the finishEdit method of the base or form feature. The base or form feature must be in an "edit" state to be able to add any additional items to it.
classType Static function that all classes support that returns the type of the class as a string. The returned string matches the string returned by the objectType property. For example if you have a reference to an object and you want to check if it's a SketchLine you can use myObject.objectType == fusion.SketchLine.classType().
item Provides access to a mesh body within the collection.

Properties

Name Description
count Returns the number of mesh bodies in the collection.
isValid Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.
objectType Returns a string indicating the type of the object.
