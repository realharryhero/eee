CopyPasteBodies Object

Derived from: Base Object

Description

Collection that provides access to all of the existing copy-paste features in a design. These are created in the UI by copying and then pasting a B-Rep body.

Defined in namespace "adsk.fusion" and the header file is "Fusion/Features/CopyPasteBodies.h".

Methods

Name Description
add Copies the specified body into the component that owns this CopyPasteBodies collection.
classType Static function that all classes support that returns the type of the class as a string. The returned string matches the string returned by the objectType property. For example if you have a reference to an object and you want to check if it's a SketchLine you can use myObject.objectType == fusion.SketchLine.classType().
item Function that returns the specified Copy/Paste Body feature using an index into the collection.
itemByName Function that returns the specified Copy/Paste Body feature using the name of the feature.

Properties

Name Description
count The number of Copy/Paste Body features in the collection.
isValid Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.
objectType Returns a string indicating the type of the object.
