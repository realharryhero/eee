BallJointMotion.rollDirectionVector Property

Parent Object: BallJointMotion

Description

Returns the direction that the roll angle is measured from. This property will return null in the case where the BallJointMotion object was obtained from a JointInput object.

Defined in namespace "adsk.fusion" and the header file is "adsk/fusion/Components/BallJointMotion.h".

# Get the value of the property.
propertyValue = ballJointMotion_var.rollDirectionVector

#include <Fusion/Components/BallJointMotion.h>

This is a read only property whose value is a Vector3D.
