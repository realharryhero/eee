RevolveFeatureInput.setTwoSideAngleExtent Method

Parent Object: RevolveFeatureInput

Description

Defines the angle of the revolve to be to applied to both sides of the profile at the specified angles.

Defined in namespace "adsk.fusion" and the header file is "adsk/fusion/Features/RevolveFeatureInput.h".

Syntax

  • C++

returnValue = revolveFeatureInput_var.setTwoSideAngleExtent(angleOne, angleTwo)

#include <Fusion/Features/RevolveFeatureInput.h>

Return Value

Type Description
boolean Returns true if successful

Parameters

Name Type Description
angleOne ValueInput The ValueInput object that defines the angle for the first side of the revolution
