MaterialLibraries.itemByName Method

Parent Object: MaterialLibraries

Description

Returns the specified Material Library using the name as seen in the user interface.

Defined in namespace "adsk.core" and the header file is "adsk/core/Materials/MaterialLibraries.h".

Syntax

Return Value

Type Description
