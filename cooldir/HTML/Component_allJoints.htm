Component.allJoints Property

Parent Object: Component

Description

Returns all joints in this component and any sub components. The joints returned are all in the context of this component so any joints in sub components will be proxies. This is primarily useful when used from the root component because Fusion 360 flattens the assembly structure, including joints, when manipulating an assembly.

Defined in namespace "adsk.fusion" and the header file is "adsk/fusion/Components/Component.h".

Syntax

This is a read only property whose value is an array of type Joint.
