UnitsManager.defaultLengthUnits Property

Parent Object: UnitsManager

Description

Returns the unit strings for the current default length unit as specified in preferences. - e.g. "cm" or "in" This is the string that is being used by Fusion 360 to represent the current length unit and is affected by the preference settings that let the user choose whether abbrevations and symbols can be used. This means that inch length units can be returned as inch, in, or ". If you need a consistent way of determing the current length unit, the distanceDisplayUnits of the FusionUnitsManager object returns an enum value.

Defined in namespace "adsk.core" and the header file is "adsk/core/Application/UnitsManager.h".
