ThreadDataQuery Object

Derived from: Base Object

Description

This object provides methods to query the thread data contained in the XML files under ThreadData folder.

Defined in namespace "adsk.fusion" and the header file is "Fusion/Features/ThreadDataQuery.h".

Methods

Name Description
allClasses Method that returns all the available classes for a thread type of a given thread designation.
allDesignations Method that returns all the available thread designations for a thread type of a given size.
allSizes Method that returns all the available thread sizes for a given thread type.
classType Static function that all classes support that returns the type of the class as a string. The returned string matches the string returned by the objectType property. For example if you have a reference to an object and you want to check if it's a SketchLine you can use myObject.objectType == fusion.SketchLine.classType().
recommendThreadData Method that gets the recommended thread data for a given model diameter.
threadTypeCustomName Method that returns the custom name for a given thread type.
threadTypeUnit Method that returns the unit for a given thread type.

Properties

Name Description
allThreadTypes Gets all the available thread types (families).
defaultInchThreadType Gets the default inch thread type.
defaultMetricThreadType Gets the default metric thread type.
isValid Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.
objectType Returns a string indicating the type of the object.
