SketchDimension Object

Derived from: Base Object

Description

The base class for the all sketch dimensions.

Defined in namespace "adsk.fusion" and the header file is "Fusion/Sketch/SketchDimension.h".

Methods

Name Description
classType Static function that all classes support that returns the type of the class as a string. The returned string matches the string returned by the objectType property. For example if you have a reference to an object and you want to check if it's a SketchLine you can use myObject.objectType == fusion.SketchLine.classType().
deleteMe Deletes this dimension. The IsDeletable property indicates if this dimension can be deleted.

Properties

Name Description
assemblyContext Returns the assembly occurrence (i.e. the occurrence) of this object in an assembly. This is only valid in the case where this is acting as a proxy in an assembly. Returns null in the case where the object is not in the context of an assembly but is already the native object.
attributes Returns the collection of attributes associated with this sketch dimension.
isDeletable Indicates if this dimension is deletable.
isDriving Gets and sets if the dimension is Driving or is Driven. Setting this property to true for a given dimension may fail if the result would overconstrain the sketch. Fusion 360 does not allow overconstrained sketches.
isValid Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.
objectType Returns a string indicating the type of the object.
parameter Returns the associated parameter or null if there is no associated parameter.
parentSketch Returns the parent sketch object.
textPosition Gets and sets position of the dimension text.
