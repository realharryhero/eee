LinearMarkingMenu.controls Property

Parent Object: LinearMarkingMenu

Description

Return the collection of top-level controls in the context menu. It's possible to have drop-down controls (fly-outs) that provide access to additional conrols. You can remove and add controls to customize the contents of the context menu.

Defined in namespace "adsk.core" and the header file is "adsk/core/UserInterface/LinearMarkingMenu.h".

Syntax

"linearMarkingMenu_var" is a variable referencing a LinearMarkingMenu object.

# Get the value of the property.
propertyValue = linearMarkingMenu_var.controls
"linearMarkingMenu_var" is a variable referencing a LinearMarkingMenu object.
#include <Core/UserInterface/LinearMarkingMenu.h>

// Get the value of the property.

Property Value

This is a read only property whose value is a ToolbarControls.
