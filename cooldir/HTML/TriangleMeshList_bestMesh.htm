TriangleMeshList.bestMesh Property

Parent Object: TriangleMeshList

Description

Returns the mesh with the tightest surface tolerance. This can return null in the case the list is empty, ie. Count is 0.

Defined in namespace "adsk.fusion" and the header file is "adsk/fusion/MeshData/TriangleMeshList.h".

# Get the value of the property.
propertyValue = triangleMeshList_var.bestMesh

Property Value

This is a read only property whose value is a TriangleMesh.
