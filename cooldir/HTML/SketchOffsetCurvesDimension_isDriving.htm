SketchOffsetCurvesDimension.isDriving Property

Parent Object: SketchOffsetCurvesDimension

Description

Gets and sets if the dimension is Driving or is Driven. Setting this property to true for a given dimension may fail if the result would overconstrain the sketch. Fusion 360 does not allow overconstrained sketches.

Defined in namespace "adsk.fusion" and the header file is "adsk/fusion/Sketch/SketchOffsetCurvesDimension.h".
