ThreadInfo Object

Derived from: Base Object

Description

This class defines the methods and properties that pertain to the thread data of a thread feature. If the ThreadInfo object is returned from a thread feature, setting this ThreadInfo will cause the thread feature to be edited.

Defined in namespace "adsk.fusion" and the header file is "Fusion/Features/ThreadInfo.h".

Methods

Name Description
classType Static function that all classes support that returns the type of the class as a string. The returned string matches the string returned by the objectType property. For example if you have a reference to an object and you want to check if it's a SketchLine you can use myObject.objectType == fusion.SketchLine.classType().

Properties

Name Description
isInternal Gets and sets if the thread is an internal or external thread. A value of true indicates an internal thread. It defaults to true.
isValid Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.
majorDiameter Gets the value that defines the major diameter.
minorDiameter Gets the value that defines the minor diameter.
objectType Returns a string indicating the type of the object.
pitchDiameter Gets the value that defines the pitch diameter.
threadAngle Gets the value that defines the thread angle.
threadClass Gets and sets the string that defines the thread class.
threadDesignation Gets and sets the string that defines the thread designation.
threadPitch Gets the value that defines the thread pitch.
threadSize Gets the string that defines the thread size.
threadType Gets and sets the string that defines the thread type.
