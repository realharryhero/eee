FavoriteMaterials Object

Derived from: Base Object

Description

Collection of the favorite materials.

Defined in namespace "adsk.core" and the header file is "Core/Materials/FavoriteMaterials.h".

Methods

Name Description
add Adds an existing material to the Favorites list
classType Static function that all classes support that returns the type of the class as a string. The returned string matches the string returned by the objectType property. For example if you have a reference to an object and you want to check if it's a SketchLine you can use myObject.objectType == fusion.SketchLine.classType().
item Returns the specified Material using an index into the collection.
itemById Returns the Material by it's internal unique ID.
itemByName Returns the specified Material using the name as seen in the user interface. This often isn't a reliable way of accessing a specific material because materials are not required to be unique.

Properties

Name Description
count The number of Materials in the collection.
isValid Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.
objectType Returns a string indicating the type of the object.
