Toolbar Object

Derived from: Base Object

Description

Provides access to a toolbar in the user interface. A toolbar is a collection of toolbar controls.

Defined in namespace "adsk.core" and the header file is "Core/UserInterface/Toolbar.h".

Methods

Name Description
classType Static function that all classes support that returns the type of the class as a string. The returned string matches the string returned by the objectType property. For example if you have a reference to an object and you want to check if it's a SketchLine you can use myObject.objectType == fusion.SketchLine.classType().

Properties

Name Description
controls Gets the controls in this toolbar.
id Gets the unique ID of the toolbar that can be used programmatically to find a specific toolbar.
isValid Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.
objectType Returns a string indicating the type of the object.
parentUserInterface Gets the owning UserInterface object.

Accessed From

Toolbars.item, Toolbars.itemById

Samples

Name Description
Write user interface to a file Writes out all of the controls in the Fusion UI to a specified text file. This is very useful when positioning your controls within existing Fusion toolbars and panels and positioning your command adjacent to an existing command.
