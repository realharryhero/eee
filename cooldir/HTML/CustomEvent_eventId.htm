CustomEvent.eventId Property

Parent Object: CustomEvent

Description

Returns the id that was assigned to this event when it was registered. Each custom event has it's own unique id.

Defined in namespace "adsk.core" and the header file is "adsk/core/Application/CustomEvent.h".

Syntax

  • Python
  • C++
"customEvent_var" is a variable referencing a CustomEvent object.

# Get the value of the property.
propertyValue = customEvent_var.eventId
"customEvent_var" is a variable referencing a CustomEvent object.
#include <Core/Application/CustomEvent.h>

// Get the value of the property.
string propertyValue = customEvent_var->eventId();

Property Value

This is a read only property whose value is a string.
