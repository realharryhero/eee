ConstructionAxisCircularFaceDefinition Object

Derived from: ConstructionAxisDefinition Object

Description

The definition for a parametric construction axis created using the SetbyCircularFace method

Defined in namespace "adsk.fusion" and the header file is "Fusion/Construction/ConstructionAxisCircularFaceDefinition.h".

Methods

Name Description
classType Static function that all classes support that returns the type of the class as a string. The returned string matches the string returned by the objectType property. For example if you have a reference to an object and you want to check if it's a SketchLine you can use myObject.objectType == fusion.SketchLine.classType().
