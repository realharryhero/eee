User Object

Derived from: Base Object

Description

A class that represents a Fusion 360 User

Defined in namespace "adsk.core" and the header file is "Core/Application/User.h".

Methods

Name Description
classType Static function that all classes support that returns the type of the class as a string. The returned string matches the string returned by the objectType property. For example if you have a reference to an object and you want to check if it's a SketchLine you can use myObject.objectType == fusion.SketchLine.classType().

Properties

Name Description
displayName Returns display name of the user. (i.e. the name that shows up in the Fusion 360 UI)
email Get the email associated with this users Fusion 360 account
isValid Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.
objectType Returns a string indicating the type of the object.
userId Returns the user's internal Autodesk account name. This can be used by applications sold through the Autodesk Exchange Store to verify that the user has in fact purchased the product.
userName Returns the Username associated with this user's Autodesk account
