Component.allAsBuiltJoints Property

Parent Object: Component

Description

Returns all joint origins in this component and any sub components. The joint origins returned are all in the context of this component so any joint origins in sub components will be proxies. This is primarily useful when used from the root component because Fusion 360 flattens the assembly structure, including joint origins, when manipulating an assembly.

Defined in namespace "adsk.fusion" and the header file is "adsk/fusion/Components/Component.h".

This is a read only property whose value is an array of type AsBuiltJoint.
