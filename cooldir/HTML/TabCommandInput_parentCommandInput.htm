TabCommandInput.parentCommandInput Property

Parent Object: TabCommandInput

Description

Gets the parent CommandInput if this commandInput is the child of a TabCommandInput or GroupCommandInput. Returns null if their is no parent.

Defined in namespace "adsk.core" and the header file is "adsk/core/UserInterface/TabCommandInput.h".

Syntax

  • Python
  • C++
"tabCommandInput_var" is a variable referencing a TabCommandInput object.

# Get the value of the property.
propertyValue = tabCommandInput_var.parentCommandInput
"tabCommandInput_var" is a variable referencing a TabCommandInput object.
#include <Core/UserInterface/TabCommandInput.h>

// Get the value of the property.
Ptr<CommandInput> propertyValue = tabCommandInput_var->parentCommandInput();

Property Value

This is a read only property whose value is a CommandInput.
