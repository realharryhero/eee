MoveFeature.faces Property

Parent Object: MoveFeature

Description

Returns the faces that were created by this feature. This works for both parametric and non-parametric features.

Defined in namespace "adsk.fusion" and the header file is "adsk/fusion/Features/MoveFeature.h".

Syntax

  • Python
  • C++
"moveFeature_var" is a variable referencing a MoveFeature object.

# Get the value of the property.
propertyValue = moveFeature_var.faces
"moveFeature_var" is a variable referencing a MoveFeature object.
#include <Fusion/Features/MoveFeature.h>

// Get the value of the property.
Ptr<BRepFaces> propertyValue = moveFeature_var->faces();

Property Value

This is a read only property whose value is a BRepFaces.
