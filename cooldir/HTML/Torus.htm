Torus Object

Derived from: Surface Object

Description

Transient torus. A transient torus is not displayed or saved in a document. A transient torus is used as a wrapper to work with raw torus information. A transient torus is a full torus with no boundaries. They are created statically using the create method of the Torus class.

Defined in namespace "adsk.core" and the header file is "Core/Geometry/Torus.h".

Methods

Name Description
classType Static function that all classes support that returns the type of the class as a string. The returned string matches the string returned by the objectType property. For example if you have a reference to an object and you want to check if it's a SketchLine you can use myObject.objectType == fusion.SketchLine.classType().
copy Creates and returns an independent copy of this Torus object.
create Creates a transient torus object.
getData Gets all of the data defining the torus.
set Sets all of the data defining the torus.
transformBy Updates this surface by transforming it with a given input matrix.

Properties

Name Description
axis Gets and sets the center axis of the torus.
evaluator Returns the surface evaluator.
isValid Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.
majorRadius Gets and sets the major radius of the torus.
minorRadius Gets and sets the minor radius of the torus.
objectType Returns a string indicating the type of the object.
origin Gets and sets the origin point (center) of the torus.
surfaceType Returns the surface type.

Accessed From
