OffsetFacesFeature.bodies Property

Parent Object: OffsetFacesFeature

Description

Returns the bodies that were modified or created by this feature. This works for both parametric and non-parametric features. For a BaseFeature this returns the bodies that are owned by the base feature.

Defined in namespace "adsk.fusion" and the header file is "adsk/fusion/Features/OffsetFacesFeature.h".

Syntax

  • Python
  • C++
"offsetFacesFeature_var" is a variable referencing an OffsetFacesFeature object.

# Get the value of the property.
propertyValue = offsetFacesFeature_var.bodies
"offsetFacesFeature_var" is a variable referencing an OffsetFacesFeature object.
#include <Fusion/Features/OffsetFacesFeature.h>

// Get the value of the property.
Ptr<BRepBodies> propertyValue = offsetFacesFeature_var->bodies();

Property Value

This is a read only property whose value is a BRepBodies.
