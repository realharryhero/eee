TextCommandPalette.isNative Property

Parent Object: TextCommandPalette

Description

Indicates if this is one of the standard Fusion 360 palettes or a custom palette created through the API. If true, it is a standard Fusion 360 palette and will have some restrictions on changing its properties and cannot be deleted.

Defined in namespace "adsk.core" and the header file is "adsk/core/UserInterface/TextCommandPalette.h".

# Get the value of the property.
propertyValue = textCommandPalette_var.isNative

#include <Core/UserInterface/TextCommandPalette.h>

Property Value

This is a read only property whose value is a boolean.
