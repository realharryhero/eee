CutPasteBody.baseFeature Property

Parent Object: CutPasteBody

Description

If this feature is associated with a base feature, this property will return that base feature. If it's not associated with a base feature, this property will return null.

Defined in namespace "adsk.fusion" and the header file is "adsk/fusion/Features/CutPasteBody.h".

Syntax

  • Python
  • C++
"cutPasteBody_var" is a variable referencing a CutPasteBody object.

# Get the value of the property.
propertyValue = cutPasteBody_var.baseFeature
"cutPasteBody_var" is a variable referencing a CutPasteBody object.
#include <Fusion/Features/CutPasteBody.h>

// Get the value of the property.
Ptr<BaseFeature> propertyValue = cutPasteBody_var->baseFeature();

Property Value

This is a read only property whose value is a BaseFeature.
