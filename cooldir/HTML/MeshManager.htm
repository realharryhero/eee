MeshManager Object

Derived from: Base Object

Description

Provides access to meshes that approximate a B-Rep and T-Spline.

Defined in namespace "adsk.fusion" and the header file is "Fusion/MeshData/MeshManager.h".

Methods

Name Description
classType Static function that all classes support that returns the type of the class as a string. The returned string matches the string returned by the objectType property. For example if you have a reference to an object and you want to check if it's a SketchLine you can use myObject.objectType == fusion.SketchLine.classType().
createMeshCalculator Creates a new MeshCalculator which is used to calculate new triangular meshes based on various parameters that control the calculation.

Properties

Name Description
displayMeshes Returns a collection that provides access to all of the existing display meshes.
isValid Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.
objectType Returns a string indicating the type of the object.
parent Returns the parent BRepBody, BRepFace, BRepLump, BRepShell, SculptBody, or SculptFace object.
