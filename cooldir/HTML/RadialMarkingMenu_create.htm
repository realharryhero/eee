RadialMarkingMenu.create Method

Parent Object: RadialMarkingMenu

Description

This is used to create a sub-menu in a marking menu. This method creates a new, empty marking menu which can then be assigned to a position in the displayed marking menu to define the sub-menu.

Defined in namespace "adsk.core" and the header file is "adsk/core/UserInterface/RadialMarkingMenu.h".

Syntax

  • Python
  • C++
"radialMarkingMenu_var" is a variable referencing a RadialMarkingMenu object.
returnValue = radialMarkingMenu_var.create(text)

#include <Core/UserInterface/RadialMarkingMenu.h>

Return Value

Type Description
RadialMarkingMenu Returns the newly created marking menu or null in the case of a failure.

Parameters

Name Type Description
text string The text that will be displayed in the parent menu to access this menu.
