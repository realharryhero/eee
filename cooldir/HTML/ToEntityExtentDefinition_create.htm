ToEntityExtentDefinition.create Method

Parent Object: ToEntityExtentDefinition

Description

Statically creates a new ToEntityExtentDefinition object. This is used as input when defining the extents of a feature to be up to a construction plane or face.

Defined in namespace "adsk.fusion" and the header file is "adsk/fusion/Features/ToEntityExtentDefinition.h".

Syntax

  • Python
  • C++
This is a static method.

# Uses no optional arguments.
returnValue = adsk.fusion.ToEntityExtentDefinition.create(entity, isChained)

# Uses optional arguments.
returnValue = adsk.fusion.ToEntityExtentDefinition.create(entity, isChained, offset)
This is a static method.

#include <Fusion/Features/ToEntityExtentDefinition.h>

// Uses no optional arguments.
returnValue = adsk::fusion::ToEntityExtentDefinition::create(entity, isChained);

// Uses optional arguments.
returnValue = adsk::fusion::ToEntityExtentDefinition::create(entity, isChained, offset);

Return Value

Type Description
ToEntityExtentDefinition Returns the newly created ToEntityExtentDefinition object or null if the creation failed.

Parameters

Name Type Description
entity Base The construction plane or face that the feature extent it up to.
isChained boolean
offset ValueInput A optional input value that defines an offset distance of the entity that will be used for the extent. Positive and negative values can be used to offset in both directions. If this argument is not provided a value of zero will be used.

This is an optional argument whose default value is null.
