ChildOperationList Object

Derived from: Base Object

Description

Provides access to the collection of child operations, folders and patterns of an existing setup.

Defined in namespace "adsk.cam" and the header file is "Cam/CAM/ChildOperationList.h".

Methods

Name Description
classType Static function that all classes support that returns the type of the class as a string. The returned string matches the string returned by the objectType property. For example if you have a reference to an object and you want to check if it's a SketchLine you can use myObject.objectType == fusion.SketchLine.classType().
item Returns the specified item using an index into the collection.
itemByName Returns the operation, folder or pattern of the specified name (the name seen in the browser).

Properties

Name Description
count Gets the number of objects in the collection.
isValid Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.
objectType Returns a string indicating the type of the object.

Accessed From

CAMFolder.children, CAMPattern.children, Setup.children
