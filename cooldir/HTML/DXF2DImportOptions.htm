DXF2DImportOptions Object

Derived from: ImportOptions Object

Description

Defines that a 2D DXF Import to create sketches (based on layers in the DXF file) is to be performed and specifies the various options.

Defined in namespace "adsk.core" and the header file is "Core/Application/DXF2DImportOptions.h".

Methods

Name Description
classType Static function that all classes support that returns the type of the class as a string. The returned string matches the string returned by the objectType property. For example if you have a reference to an object and you want to check if it's a SketchLine you can use myObject.objectType == fusion.SketchLine.classType().

Properties

Name Description
filename Gets and sets the filename or url of the file to be imported.
isValid Indicates if this object is still valid, i.e. hasn't been deleted or some other action done to invalidate the reference.
isViewFit Specifies if the camera should be adjusted to fit the geometry of the import. This defaults to true, which will cause a change in the current view. Setting this to false will leave the view as-is and just import the geometry.
objectType Returns a string indicating the type of the object.
planarEntity Gets and sets the construction plane or planar face that defines the plane that the resulting sketches will be created on.
position Gets and sets the X,Y offset position for the origin of the imported DXF data relative to the sketch origin. This defaults to (0,0) in a newly created DXF2DImportOptions object.
results Returns a collection of Sketch objects. A sketch is created for each layer in the dxf file that contains 2D geometry. Any 3D geometry contained in the DXF file is ignored. The names of the resulting sketches correspond to the layer names in the DXF file. Currently, the only way to get a single sketch as a result is to supply a DXF file that only has 2D geometry on a single layer.

Accessed From
