Curve2D.evaluator Property

Parent Object: Curve2D

Description

Returns an evaluator object that lets you perform additional evaluations on the curve.

Defined in namespace "adsk.core" and the header file is "adsk/core/Geometry/Curve2D.h".

Syntax

  • Python
  • C++
"curve2D_var" is a variable referencing a Curve2D object.

# Get the value of the property.
propertyValue = curve2D_var.evaluator
"curve2D_var" is a variable referencing a Curve2D object.
#include <Core/Geometry/Curve2D.h>

// Get the value of the property.
Ptr<CurveEvaluator2D> propertyValue = curve2D_var->evaluator();

Property Value

This is a read only property whose value is a CurveEvaluator2D.
